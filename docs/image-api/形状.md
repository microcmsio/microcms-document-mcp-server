# 形状

パラメータ
=====

mask=corners
------------

画像を角丸の形状に変更できます。  
  
例）_?mask=corners_



corner-radius
-------------

`mask=corners`を指定した場合に、角丸の半径をピクセル単位で指定できます。  
  
例）_?mask=corners&corner-radius=10_



mask=ellipse
------------

画像を楕円形に変更できます。  
縦幅と横幅が同じ場合は円形となります。  
  
例）_?mask=ellipse_
