# フォーマット

パラメータ
=====

fm=webp
-------

画像をWebP形式に変換します。

### WebPとは？

Googleが開発している比較的新しい画像フォーマットです。「ウェッピー」と読みます。  
通常のjpgやpngをこのwebp形式に変換をかけるだけで、20%~30%ほどサイズが小さくなる場合があります。

以下、PNG形式の元画像に `?fm=webp` を付与し、Webp形式に変換した例です。  
  
**▼URL**  
[https://images.microcms-assets.io/assets/d6af1616730544a596d299c20834f460/0b33104ab3fd2ba2d6b12e8ef25734de/sample-02.png?fm=webp](https://images.microcms-assets.io/assets/d6af1616730544a596d299c20834f460/0b33104ab3fd2ba2d6b12e8ef25734de/sample-02.png?fm=webp)



fm=avif
-------

画像をAVIF形式に変換します。

### AVIFとは？

GoogleやAmazon、Netflixなどが参加する企業連合が開発している新しい画像フォーマットです。  
WebP形式よりも高い圧縮率を期待できますが、最新フォーマットのため、古いバージョンのブラウザでは非対応の場合があります。

以下、PNG形式の元画像に `?fm=avif` を付与し、AVIF形式に変換した例です。  
  
**▼URL**  
[https://images.microcms-assets.io/assets/d6af1616730544a596d299c20834f460/0b33104ab3fd2ba2d6b12e8ef25734de/sample-02.png?fm=avif](https://images.microcms-assets.io/assets/d6af1616730544a596d299c20834f460/0b33104ab3fd2ba2d6b12e8ef25734de/sample-02.png?fm=avif)



fm=jpg
------

画像をJPEG形式に変換します。

fm=png
------

画像をPNG形式に変換します。
