# ウォーターマーク

パラメータ
=====

mark
----

ウォーターマークとして使用する画像のURLを指定し、Z画像にウォーターマークを追加できます。  
指定する画像のURLには、画像APIのパラメータを使用できます。  
  
例）_?mark=https://example.com/watermark.png_



mark-alpha
----------

ウォーターマークの透過率を指定できます。  
  
例）_?mark-alpha=60&mark=https://example.com/watermark.png_



mark-align
----------

ウォーターマークの配置場所を指定できます。デフォルトはbottomとrightの指定となります。

*   top: ベース画像の上部に配置します。
*   middle: ベース画像の垂直中央に配置します。
*   bottom: ベース画像の下部に配置します。
*   left: ベース画像の左側に配置します。
*   center: ベース画像の水平中央に配置します。
*   right: ベース画像の右側に配置します。

  
例）_?mark-align=top,left&mark=https://example.com/watermark.png_
