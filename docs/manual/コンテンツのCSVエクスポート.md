# コンテンツのCSVエクスポート

管理画面のコンテンツ一覧から、コンテンツデータをCSVファイルとしてエクスポートできます。簡易的なバックアップ、Excelでの編集・分析、既存データを編集して再インポートなどの用途に活用できます。

Teamプラン、Businessプラン、Enterpriseプランのお客様にご利用いただける機能です。  
プランごとに利用できる機能については、[料金プランページ](https://microcms.io/pricing)をご覧ください。

操作手順
====

１：コンテンツ一覧画面を開きます。  
２：エクスポート対象のビューを選択します。

*   **全件エクスポートする場合**：「全てのコンテンツ」のビューのまま、次の手順に進みます。
*   **特定のデータのみを抽出したい場合**：ビュー機能で設定したタブを選択して、次の手順に進みます。

![](https://images.microcms-assets.io/assets/d6af1616730544a596d299c20834f460/72ce752dfbf64cbf95831657c31e1b54/CleanShot%202026-05-07%20at%2017.16.385.png)  
  
ビュー機能の詳細ついては[ドキュメント](https://document.microcms.io/manual/contents-view)をご参照ください。  
  
３：**［CSVエクスポート］** ボタンをクリックします。  
![](https://images.microcms-assets.io/assets/d6af1616730544a596d299c20834f460/45679518394a48fda5b6e877fd51a84a/csv-export-flow02.png)  
  
４：エクスポートに関するオプションを確認・設定するダイアログが表示されます。エクスポート名を任意に設定できます。  
**エクスポート対象のデータは、現在のビューで表示されている条件（フィルター・キーワード検索・ソート）に基づきます。**  
![](https://images.microcms-assets.io/assets/d6af1616730544a596d299c20834f460/8392e4de22934739bf7fa0e7a5b64891/csv-export-flow03.png)  
  
５：内容を確認し、**［エクスポートを開始］** ボタンをクリックします。  
![](https://images.microcms-assets.io/assets/d6af1616730544a596d299c20834f460/5fb909106bf94841b14b8482efe55683/csv-export-flow04.png)  
  
６：「CSVエクスポートを開始しました」というトースト通知が表示されます。  
７：エクスポートが完了すると、指定のメールアドレス宛にダウンロードリンクが記載されたメールが届きます。また、管理画面上にもトースト通知が表示されます。  
![](https://images.microcms-assets.io/assets/d6af1616730544a596d299c20834f460/6cb9badb4b1a477cbd41996e82900428/CleanShot%202026-05-07%20at%2016.41.065.png)  
  
８：メール内のダウンロードリンクをクリックして、CSVファイルをダウンロードします。  
![](https://images.microcms-assets.io/assets/d6af1616730544a596d299c20834f460/b538461416894b29a57397289e6f48ed/csv-export-flow06.png)

*   ダウンロードリンクは、エクスポートを実行したユーザーのみが使用できます。
*   ダウンロードリンクの有効期限は**24時間**です。期限を過ぎるとダウンロードできなくなります。有効期限が切れた場合は、再度エクスポートを実行してください。

データ仕様・フォーマット
============

共通カラム
-----

すべてのCSVファイルには、以下の共通カラムが含まれます。

カラム名

説明

形式

備考

status

コンテンツのステータス

`PUBLISH` / `DRAFT` / `CLOSED`

「公開中かつ下書き中」のコンテンツは、公開中（`PUBLISH`）と下書き中（`DRAFT`）の2行に分かれて出力されます。

id

コンテンツID

コンテンツに設定されたコンテンツIDの文字列

\-

createdBy

作成者

`メンバーの表示名` / `CSV_IMPORT` / `API` / `REMOVED_FROM_SERVICE`

*   `メンバーの表示名`：表示名が未入力の場合はメールアドレスの@の手前部分が出力されます（例：user1@example.com → user1）
*   `CSV_IMPORT`：CSVインポートによる登録
*   `API`：コンテンツAPIによる登録
*   `REMOVED_FROM_SERVICE`：サービスから削除済みのメンバーによる登録

updatedBy

更新者

作成者と同様

\-

createdAt

作成日時

ISO 8601形式（例：`2026-12-10T05:10:37.051Z`）

\-

updatedAt

更新日時

作成日時と同様

\-

publishedAt

公開日時

作成日時と同様

\-

revisedAt

改定日時

作成日時と同様

\-

各フィールドのカラム
----------

各フィールドのデータが出力されます。データの形式は原則としてCSVインポートで用いられる入力形式と同じです。

https://document.microcms.io/manual/csv-import#had265a33b7

制限／注意事項
=======

権限について
------

*   エクスポート対象のAPIおよびコンテンツの読み取り権限が必要です。
*   エクスポートするコンテンツが参照しているコンテンツやメディア、メンバーの読み取り権限がない場合は「`NO_READ_PERMISSION`」と出力されます。

その他の注意点
-------

*   コンテンツ数が多い場合や、データ構造が複雑な場合は、エクスポートの完了までに30分以上かかることがあります。
*   エクスポート実行中に行われたコンテンツやAPIスキーマの編集は、エクスポート結果に反映される場合があります。正確なデータが必要な場合は、編集作業を控えた状態でエクスポートを実行してください。
*   生成されるCSVのファイル名は `{エクスポート名}-{APIエンドポイント}-{タイムスタンプ}.csv` の形式になります。

よくある質問
======

**Q.** **エクスポート完了のメールが届かない場合は？**  
A. 迷惑メールフォルダや別フォルダに振り分けされていないかご確認ください。また、エクスポートの完了には、コンテンツ数やフィールドの構成によって時間がかかる場合があります。しばらく待ってから再度ご確認ください。  
  
**Q. Excelで開くと文字化けする場合は？**  
文字化けした際の対処法については、ヘルプ「[CSVインポート用のサンプルファイルを開くと文字化けします。対処法は？](https://help.microcms.io/ja/knowledge/csv-character-code)」をご確認ください。  
  
**Q. エクスポートボタンが表示されません**  
Hobbyプランではご利用いただけません。CSVエクスポートはTeamプラン、Businessプラン、Enterpriseプランでご利用いただける機能です。  
  
**Q.  エクスポートしたCSVを再インポートするには？**  
再インポートする場合は、エクスポートしたCSVファイルから**id以外**の[共通カラム](#he50444d968)を削除した上で、[コンテンツのCSVインポート](https://document.microcms.io/manual/csv-import)を実行してください。  
なお、再インポートは、同様のAPIスキーマに対してコンテンツを新規登録する用途を想定しています。

エクスポートしたCSVをインポートしても、元のコンテンツを完全に復元できるとは限りません。以下のようなケースでは、インポートできない場合があります。

*   CSVインポートで対応していないフィールドや値の形式が含まれている場合
*   権限不足により、一部のフィールドの値を取得できない状態でエクスポートした場合
*   同一コンテンツについて、公開中（`PUBLISH`）と下書き中（`DRAFT`）の2行がCSVに含まれている場合
*   既存コンテンツがあるAPIにそのままインポートし、コンテンツIDやユニーク制約のあるフィールドが重複する場合

インポートの仕様については、ドキュメント「[コンテンツのCSVインポート](https://document.microcms.io/manual/csv-import)」をご参照ください。
